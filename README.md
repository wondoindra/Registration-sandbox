# Simple Signup API

## Tech used
- Express.js
- Postgresql
- Sequelize
- Nodemailer with Mailtrap
---

## Hosted at Heroku
> METHOD: POST
>
> URL: https://incit-be.herokuapp.com/user/signup
>
> Body: x-www-form-urlencoded
>
> Payload: Email and Password

> METHOD: GET
>
> URL: https://incit-be.herokuapp.com/user/:id
---
